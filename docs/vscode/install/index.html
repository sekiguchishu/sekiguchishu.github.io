VSCodeのインストール

ダウンロード

　まずはダウンロードします。「vscode」で検索し，「Visual Studio Code - Code Editing. Redefined」というタイトルのページを開いてください。

「Download for ●●●」をクリックします。 このようなページに切り替わったら，自動的にダウンロードが始まります。

インストール

　Macの場合はダウンロードしたファイルを解凍し，できあがったアプリケーションファイルをアプリケーションフォルダに入れておき，クリックすれば起動できます。
　Windowsの場合はダウンロードしたexeファイルを起動します。普通に起動すると，インストール先を「Program Files」にしたい場合，アクセスが拒否されるので，exeファイルを右クリックし，「管理者として実行」します。

このような表示が出ると思いますがOKを押します。あとはインストーラの指示に従うだけです。以下に想定される表示を載せておきます。
