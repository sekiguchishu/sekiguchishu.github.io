注意事項

　TeXを使ううえでの注意事項を，重要な順に列挙します。

  • ファイル名・フォルダ名は，半角英数字とアンダーバーだけにすべし
    ドットtexファイル本体の名前だけでなく，それに至るまでのフルパスにも，日本語は使ってはいけません。例えば，Windowsの場合，
    “C:\Users\shu\Desktop\卒論\template\tex\sotsuron_main.tex”
    のように，途中のフォルダ名に入っていてもだめです。また，図として挿入するPDFファイル名にも注意してください。日本語がだめなのはもちろん，「ドット」や「半角スペース」もだめです。ただし拡張子のドットは大丈夫です。
  • ベタ書き禁止，Figref，Tabrefを使うべし
    TeXのソースコード内で，図~\ref{ラベル名}やTable.~\ref{ラベル名}のように直接書かずに，\Figref{ラベル名}や\Tabref{ラベル名}と書くようにしましょう。他にも，\Equref（式），\Chapref（章），\Secref（節），\Subsecref（項），\Subsubsecref（目）があります。また，図のサイズや余白を調整する際，「センチメートル」や「pt」などの絶対量で指定するのではなく，\textwidthや\columnwidthなどの相対量を使いましょう。
